# PdfInput.TextLinesAsync method (2 of 2)

Gets the lines of text for this [`PdfInput`](../PdfInput.md) asynchronously, optionally you can set both the start and end pages and a value indicating whether blank lines are included in the result.

```csharp
public Task<IEnumerable<PdfTextLine>> TextLinesAsync(int? fromPage = null, int? toPage = null, 
    bool removeEmptyLines = true, CancellationToken cancellationToken = default)
```

| parameter | description |
| --- | --- |
| fromPage | Defines start page. If a value is not set, it will default to 1 |
| toPage | Defines end page. If a value is not set, it will default to total document pages |
| removeEmptyLines | Indicates whether blank lines are included in the result. By default they are not included |
| cancellationToken | A cancellation token that can be used by other objects or threads to receive notice of cancellation. |

## Exceptions

| exception | condition |
| --- | --- |
| ArgumentException | If document has no pages |
| ArgumentOutOfRangeException | If *fromPage* is less than one or is greater than the total number of pages of the document |
| ArgumentOutOfRangeException | If *toPage* is less than one or is greater than the total number of pages of the document |
