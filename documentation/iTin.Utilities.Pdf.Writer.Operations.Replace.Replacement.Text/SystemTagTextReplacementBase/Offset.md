# SystemTagTextReplacementBase.Offset property

Gets or sets a reference a point structure which represents the text/image/table offset. The default is Empty. Positive values on the y axis move the text/image/table down and positive values on the x axis move the text/image right.

```csharp
public PointF Offset { get; set; }
```

## Property Value

A PointF object that contains text/image/table offset to apply.
